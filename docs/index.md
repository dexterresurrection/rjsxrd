# rjsxrd

Автоматически обновляемая коллекция публичных VPN-конфигураций для обхода блокировок.

**Репозиторий:** [github.com/whoahaow/rjsxrd](https://github.com/whoahaow/rjsxrd)  
**Telegram-канал:** [t.me/rjsxrd](https://t.me/rjsxrd)

## О проекте

rjsxrd собирает, фильтрует и верифицирует тысячи VPN-конфигов из открытых источников. Результат — файлы, готовые к импорту в любой современный VPN-клиент.

**Ключевые характеристики:**

- **Обновление:** ежечасно с VPS через cron (основной) + каждые 48 часов через GitHub Actions (резервный)
- **Протоколы:** VLESS, VMess, Trojan, Shadowsocks, ShadowsocksR, Hysteria v1/v2, TUIC
- **Верификация:** Xray-core или TCP-пинг, сортировка по задержке
- **Фильтрация:** автоматическое удаление небезопасных и нерабочих конфигов
- **SNI/CIDR:** специальные наборы для обхода мобильных белых списков
- **Telegram-прокси:** сбор и верификация MTProto и SOCKS5 прокси

## Быстрая навигация по файлам

```
githubmirror/
├── default/             # Основные конфиги (1.txt, all.txt, all-secure.txt)
├── bypass/              # SNI/CIDR-фильтрованные, безопасные
├── bypass-unsecure/     # SNI/CIDR-фильтрованные, все (включая небезопасные)
├── split-by-protocols/  # Разделение по протоколам (vless.txt, vmess.txt, ...)
└── tg-proxy/            # Telegram-прокси (all.txt, MTProto.txt, socks.txt)
```
