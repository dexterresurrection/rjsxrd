# Быстрый старт

Два шага, чтобы начать пользоваться конфигами.

## Шаг 1. Скопируйте ссылку

Рекомендуемые файлы для начала:

| Файл | Ссылка |
|------|--------|
| 1.txt | `https://raw.githubusercontent.com/whoahaow/rjsxrd/refs/heads/main/githubmirror/default/1.txt` |
| bypass-all.txt | `https://raw.githubusercontent.com/whoahaow/rjsxrd/refs/heads/main/githubmirror/bypass/bypass-all.txt` |
| all-secure.txt | `https://raw.githubusercontent.com/whoahaow/rjsxrd/refs/heads/main/githubmirror/default/all-secure.txt` |

**Какой выбрать?**
- Для мобильного интернета с белыми списками (Россия, Узбекистан и др.) — `bypass/bypass-all.txt`
- Для обычных блокировок — любой файл из `default/`, например `6.txt`, `22.txt`, `23.txt`
- Для минимального риска — `all-secure.txt` (только безопасные конфиги)

Все ссылки собраны в разделе [Генерируемые файлы](user/config-files.md).

## Шаг 2. Импортируйте в клиент

Скопируйте ссылку в буфер обмена и откройте ваш VPN-клиент:

- **Android (v2rayNG):** `+` → «Импорт из буфера обмена»
- **Windows (Throne):** «Профили» → «Добавить профиль из буфера обмена»
- **iOS (V2Box):** «Config» → `+` → «Добавить подписку»
- **macOS (Hiddify):** «Новый профиль» → «Добавить из буфера обмена»

Подробные инструкции с картинками — в [Импорт в клиенты](user/import-guide.md).

После импорта выберите сервер с минимальным пингом и подключитесь.
